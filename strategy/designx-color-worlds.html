The Stack · AI Tools

THE STACK
COLOR WORLDNEW

The Stack needs its own identity — it's DesignX's tool curation platform. It can't share purple (that's DLS premium) or blue (that's jobs). It needs something that says "curated tech picks" without overlapping existing worlds.

Three Options for The Stack's Color World
The Stack sits between "editorial curation" and "tech tools." It needs to feel distinct from the 5 existing worlds while still belonging to the DesignX family.
Option A: Dark Navy → Crimson
Cinematic, editorial. The dark navy base is unique to this world, crimson endpoint is distinct from both coral and red-for-errors. Feels "curated picks" — like a film critic's favorites list.
★ Recommended
Option B: Deep Green → Teal
Fresh, dev-adjacent. Teal sits between jobs-blue and salary-green without colliding with either. Feels like "code + design." Risk: might read too close to salary green at small scale.
Option C: Dark Copper → Amber
Metallic, premium tools vibe. Problem: overlaps significantly with Community coral world's brown and amber tones. Would need to steal from community.
If We Go With Option A (Crimson)
Hero gradient135deg · #1a1a2e → #0f3460 → #e94560
Subtle gradient135deg · crimson 10% → rose 5%
Spot color (primary)#e94560 — links, badges, active states
Spot color (light)#f58b8b — hover, pale accents
Duotone overlay#e94560 at 0.85 opacity, multiply blend
Collision with redCrimson ≠ error red. #e94560 is pink-red, #EF4444 is pure red. Distinct.

Usage Rules

WHEN TO USE
WHAT

Not just "which gradient" — which gradient treatment at which intensity on which surface. This is the decision tree for every color situation.

Gradient Treatment Ladder — Every World Gets These
From most intense to most subtle:
1. Hero GradientFull saturation · 135deg · Card headers, page heroes, section banners
2. Text Gradientbackground-clip:text · Display type 32px+ only · Never body or UI text
3. Gradient Border1px stroke via padding trick · Featured cards, selected states, premium CTAs
4. GlowRadial, blur(60px), 15% opacity · Ambient light behind key content
5. Subtle Gradient5-12% opacity · Section washes, hover backgrounds, selected rows
6. Spot ColorSingle hex value · Links, badges, icons, status indicators · Works at 12px

Gradient Direction Rules
135deg — Default
Card headers, hero banners, section backgrounds. The diagonal creates dynamism and is the DesignX signature angle.
90deg — Horizontal
Nav accents, stat bars, horizontal dividers, wide thin strips. Use when the element is significantly wider than tall.
180deg — Vertical
Mobile stories, vertical cards, sidebar accents. Use when the element is taller than wide.
Hard Rules
Never Do:
Mix two product gradients on one surface — every element belongs to one world. Use hero gradient as body text background — it overwhelms content. Put gradient text under 32px — it becomes unreadable. Use spot color for large fills — that's what the gradient is for. Create new gradients per campaign — use the season flex for that. Use #EF4444 red as a content color — it's reserved for errors/destructive.
Collision Rules

WHEN WORLDS
OVERLAP

Salary data on the job board. DLS talks on the homepage. Community photos in the newsletter. When two product worlds show up in the same space, which one wins?

The Hierarchy
Page context wins. The page determines the gradient world — elements within it use their product's spot color only.
Salary card on Jobs pageCard uses green spot color text. Page hero stays blue.
Job listing with salarySalary figure = neon green (JetBrains Mono). Job card frame = blue world.
DLS talk on homepageTalk card uses DLS event graphic. Homepage hero = Season '26.
Community photo in newsletterPhoto gets coral duotone. Newsletter header = Season '26.
The Stack tool on homepageStack card uses crimson spot color. Homepage context = Season '26.
Cross-Product Rule
Spot Color = Always Allowed
Any product's spot color can appear anywhere as text, icon, or badge color. Neon salary figures on blue job cards. Purple DLS badges on coral community pages. Spot colors are the "passport" — they travel freely.
Hero Gradient = Page Owner Only
Only the page's owning product world gets to use its hero gradient as a fill. You never see a full green gradient block inside a blue job board page. The gradient is territorial.
